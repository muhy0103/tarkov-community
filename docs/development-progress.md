# 开发进度记录

更新时间：2026-06-04

## 项目定位

本项目是“逃离塔科夫玩家情报社区管理系统”。整体定位为玩家社区论坛，而不是资料百科站。页面设计以明亮、清爽、功能分明为主，论坛内容围绕地图、任务、装备弹药、市场经济、藏身处、Boss 情报、战局复盘和组队招募展开。

- Vue 3 + Vite 工程，使用 JavaScript

- Pinia 用户登录态管理
- Axios 请求封装
- Element Plus 组件库
- 首页社区概览：

  - 支持进入帖子详情

- 情报广场：
  - 支持集中浏览帖子列表

- 帖子详情页：
  - 展示帖子内容、作者、分区、类型、浏览量、评论数
  - 支持发表评论
  - 支持点赞
  - 支持收藏
- 发布情报页：
  - 登录用户可选择分区和帖子类型
  - 可填写标题、正文并创建帖子
  - 创建成功后自动跳转帖子详情页
  - 未登录用户不能提交

## 当前注意点

- 认证功能目前是课程项目演示级 token，后续可升级为 JWT 或 Spring Security 完整鉴权链路。
- 前端当前使用 JavaScript，不使用 TypeScript，符合项目要求。
- 数据库表结构较完整，但演示数据仍然较少，后续可以根据页面和答辩需要补充。
- 前端页面保持清爽社区风格，不走传统塔科夫 wiki 的密集资料堆叠路线。
