# 开发进度记录

更新时间：2026-06-05

## 项目定位

本项目是“逃离塔科夫玩家情报社区管理系统”。整体定位为玩家社区论坛，而不是资料百科站。页面设计以明亮、清爽、功能分明为主，论坛内容围绕地图、任务、装备弹药、市场经济、藏身处、Boss 情报、战局复盘和组队招募展开。

### 前端

- Vue 3 + Vite 工程，使用 JavaScript
- Vue Router 页面路由
- 后台路由权限控制：
  - 未登录访问后台页面时跳转登录页
  - 登录后可回到原目标页面
  - 普通用户访问后台页面时进入无权限提示页
- Pinia 用户登录态管理
- Axios 请求封装
- Element Plus 组件库
