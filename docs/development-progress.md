# 开发进度记录

更新时间：2026-06-05

## 项目定位

本项目是“逃离塔科夫玩家情报社区管理系统”。整体定位为玩家社区论坛，而不是资料百科站。页面设计以明亮、清爽、功能分明为主，论坛内容围绕地图、任务、装备弹药、市场经济、藏身处、Boss 情报、战局复盘和组队招募展开。

- 建立项目目录结构：`frontend`、`backend`、`database`、`docs`
- 初始化 Git 仓库并连接 GitHub：`muhy0103/tarkov-community`
- 完成完整需求和四周构建计划文档

### 前端

- Vue 3 + Vite 工程，使用 JavaScript
- Vue Router 页面路由
- 后台路由权限控制：
  - 未登录访问后台页面时跳转登录页
  - 登录后可回到原目标页面
  - 普通用户访问后台页面时进入无权限提示页
- Pinia 用户登录态管理
- Axios 请求封装
- Element Plus 组件库
- 首页社区概览：
  - 展示分区、地图、商人、任务、装备弹药、Boss、藏身处数据
  - 展示最新帖子列表
  - 支持进入帖子详情
  - 支持进入发布情报页
- 情报广场：
  - 支持集中浏览帖子列表
  - 支持关键词搜索
  - 支持按分区、类型和推荐状态筛选
  - 支持分页浏览
- 登录注册页：
  - 注册后自动进入登录态
  - 登录后保存 token 和用户信息
  - 顶部导航显示用户昵称和退出按钮
- 帖子详情页：
  - 展示帖子内容、作者、分区、类型、浏览量、评论数
  - 支持发表评论
  - 支持点赞
  - 支持收藏
  - 评论、点赞和收藏请求不再向后端提交 `userId`
- 发布情报页：
  - 登录用户可选择分区和帖子类型
  - 可填写标题、正文并创建帖子
  - 创建成功后自动跳转帖子详情页
  - 未登录用户不能提交
  - 发帖请求不再向后端提交 `userId`
- 用户中心页：
  - 接入当前登录用户个人统计接口
  - 支持查看我的帖子、我的评论和我的收藏
  - 使用标签页区分内容类型，避免页面信息堆叠
  - 用户中心拉到最新个人资料后会同步刷新前端登录态昵称
  - 支持在用户中心弹窗编辑昵称、邮箱和头像
  - 保存资料成功后会同步刷新页面顶部资料和前端登录态
- 后台概览页：
  - 接入后台概览统计接口
  - 展示用户、帖子、评论、分区和资料库统计
  - 预留帖子审核、用户管理、资料维护入口
- 后台帖子审核页：
  - 接入后台帖子审核列表接口
  - 支持按状态、分区和关键词筛选帖子
  - 支持隐藏、恢复、推荐、取消推荐、置顶和取消置顶
  - 使用表格集中展示帖子状态、标记、互动数和发布时间
- 后台用户管理页：
  - 接入后台用户管理列表接口
  - 支持按角色、状态和关键词筛选用户
  - 支持设为版主、恢复普通、禁用和恢复账号
  - 使用表格集中展示用户昵称、账号、邮箱、角色、状态、贡献值和时间信息
- 后台评论审核页：
  - 接入后台评论审核列表接口
  - 支持按状态和关键词筛选评论
  - 支持隐藏和恢复评论
  - 使用表格集中展示评论内容、所属帖子、作者、状态、互动数和发布时间
